# تصميم موقع مصنع نعيم - فلسفة التصميم

## Design Movement: Minimalist Geometric Luxury
تصميم هندسي بسيط مع لمسات فاخرة، يعكس جودة وأناقة منتجات الرخام والجرانيت.

## Core Principles
1. **Whitespace as Design**: استخدام فراغات بيضاء واسعة لإعطاء الموقع شعوراً معمارياً فاخراً
2. **Geometric Precision**: استخدام أشكال هندسية نظيفة وخطوط واضحة
3. **Material Authenticity**: إظهار جمال المواد الطبيعية من خلال الصور عالية الجودة
4. **Premium Restraint**: استخدام محدود للألوان والحركة لتعزيز الفخامة

## Color Philosophy
- **Primary Background**: أبيض نقي (#FFFFFF) ورمادي فاتح جداً (#F8F7F5)
- **Text**: فحم عميق جداً (#1A1A1A) و رمادي داكن (#2D2D2D)
- **Accent**: ذهب معدني (#D4AF37) و نحاس (#B8860B) للـ hover states
- **Reasoning**: الألوان الترابية والذهبية تعكس طبيعة المواد الفاخرة (الرخام والجرانيت)

## Layout Paradigm
- **Asymmetric Grid**: استخدام تخطيط غير متماثل يعكس الطبيعة الحرة للحجر
- **Vertical Flow**: تدفق عمودي واضح مع فواصل بصرية هندسية
- **Hero-First**: صورة بطل قوية تحتل المساحة الكاملة في الأعلى

## Signature Elements
1. **Geometric Dividers**: فواصل هندسية بسيطة بين الأقسام (خطوط قطرية، أشكال بسيطة)
2. **Gold Accents**: لمسات ذهبية على الأزرار والـ hover states
3. **Product Cards**: بطاقات منتجات بسيطة مع ظلال ناعمة وحدود رقيقة

## Interaction Philosophy
- **Subtle Transitions**: انتقالات سلسة عند التفاعل (0.3s ease-out)
- **Hover Elegance**: تغيير لون الذهب عند التمرير فوق العناصر
- **Language Toggle**: تبديل سلس بين العربية والإنجليزية مع تغيير اتجاه الصفحة

## Animation Guidelines
- **Entrance Animations**: fade-in بطيء (0.6s) عند تحميل الصفحة
- **Scroll Animations**: fade-up بسيط عند الوصول للعنصر
- **Interactive Feedback**: تغيير طفيف في الحجم (scale 1.02) عند التمرير
- **No Distraction**: تجنب الحركات المفرطة التي تشتت الانتباه

## Typography System
- **Arabic Font**: Cairo (وزن 400, 600, 700) - خط عربي أنيق وقابل للقراءة
- **English Font**: Roboto (وزن 300, 400, 600, 700) - خط حديث وواضح
- **Hierarchy**:
  - H1: Cairo/Roboto 700, 48px (Desktop) / 36px (Mobile)
  - H2: Cairo/Roboto 600, 36px (Desktop) / 28px (Mobile)
  - Body: Cairo/Roboto 400, 16px
  - Small: Cairo/Roboto 400, 14px
